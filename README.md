# LimeLight Products Overview

## Build Setup

```bash
# Install dependencies
$ yarn install

# Serve with hot reload at localhost:3000
$ yarn dev

# Build for production and launch server
$ yarn build
$ yarn start

# Lints and fixes files
$ yarn lint

# Generate static project
$ yarn generate
```
